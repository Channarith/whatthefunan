WHAT THE FUNAN - LEGAL COMPLIANCE CHECKLIST

PRE-DEVELOPMENT PHASE
================================================================================

[ ] Brief ALL team members on compliance rules
[ ] Set up compliance review workflow in project management
[ ] Create content approval process with required sign-offs

================================================================================
CONTENT CREATION PHASE
================================================================================

CHARACTER DESIGN REVIEW
--------------------------------------------------------------------------------
[ ] All character names are FICTIONAL (no historical rulers)
[ ] No resemblance to Thai royal family members (any era)
[ ] No monks as playable or combat characters
[ ] Apsara designs are respectful, not sexualized
[ ] Animal characters avoid ethnic stereotypes
[ ] No ethnic caricatures or offensive designs
[ ] All character designs reviewed by cultural consultants

ENVIRONMENT ART REVIEW
--------------------------------------------------------------------------------
[ ] All temples are FICTIONAL "Funan-inspired" designs
[ ] NO direct replicas of Angkor Wat
[ ] NO replicas of Preah Vihear Temple
[ ] NO replicas of any Thai royal palaces
[ ] Buddha statues shown in respectful contexts ONLY
[ ] Buddha images NOT in combat zones or battle arenas
[ ] NO destruction of sacred architecture in gameplay
[ ] All environments reviewed by cultural consultants

NARRATIVE & DIALOGUE REVIEW
--------------------------------------------------------------------------------
[ ] All king/ruler names are FICTIONAL
[ ] NO references to Thai royal family (any era) - CRITICAL
[ ] NO references to Khmer Rouge or genocide
[ ] NO references to Cambodia-Thailand border disputes
[ ] NO political commentary about any country
[ ] Cultural elements framed as "Funan" (shared heritage)
[ ] NO claims of cultural origin favoring one country
[ ] Educational content is factually accurate
[ ] All narrative reviewed by cultural consultants

RELIGIOUS CONTENT REVIEW
--------------------------------------------------------------------------------
[ ] Buddha images ALWAYS respectful
[ ] NO Buddha images in combat zones
[ ] NO Buddha images being damaged or destroyed
[ ] NO monks in violent or inappropriate situations
[ ] Hindu deities depicted respectfully
[ ] Nagas/dragons depicted as mythological (SAFE)
[ ] Religious ceremonies shown respectfully
[ ] NO mockery of religious practices
[ ] All religious content reviewed by consultants

GAMEPLAY MECHANICS REVIEW
--------------------------------------------------------------------------------
[ ] NO real-money gambling mechanics
[ ] NO gacha system with real money purchases
[ ] Virtual currency has clear "no real value" disclaimer
[ ] NO slot machines or casino simulations
[ ] Parental gate implemented for ALL purchases
[ ] Age gate implemented at first launch
[ ] Combat does NOT target religious figures or sacred sites

AUDIO CONTENT REVIEW
--------------------------------------------------------------------------------
[ ] Traditional music used respectfully
[ ] NO sacred Buddhist chants used inappropriately
[ ] Voice acting avoids offensive accents/stereotypes
[ ] NO offensive stereotypical portrayals

UI/UX & TEXT REVIEW
--------------------------------------------------------------------------------
[ ] Legal disclaimers included in game loading screen
[ ] Privacy policy accessible from main menu
[ ] Terms of service accessible from main menu
[ ] Age-appropriate content warnings displayed
[ ] Parental controls easily accessible
[ ] All disclaimers reviewed by legal counsel

================================================================================
DATA PROTECTION COMPLIANCE
================================================================================

COPPA (Children Under 13)
--------------------------------------------------------------------------------
[ ] No data collection from children under 13 without parental consent
[ ] Verifiable parental consent system implemented
[ ] Privacy policy specifically addresses children
[ ] No behavioral advertising to children
[ ] Data deletion capability for parent requests
[ ] Annual COPPA compliance audit scheduled

GDPR (European Union)
--------------------------------------------------------------------------------
[ ] Clear consent before data collection
[ ] Right to access personal data implemented
[ ] Right to data deletion implemented
[ ] Right to data portability implemented
[ ] Data processing documentation maintained
[ ] DPO (Data Protection Officer) designated if required

THAILAND PDPA
--------------------------------------------------------------------------------
[ ] Obtain explicit consent before collecting personal data
[ ] For users under 20: require parental/guardian consent
[ ] Clear privacy policy in Thai language
[ ] Data localization requirements reviewed
[ ] Cross-border data transfer compliance verified
